# 🚖 OLA Data Analytics Project

## 📌 Project Overview

This project is an end-to-end **OLA Ride Booking Data Analytics Dashboard** built using **SQL** and **Power BI**.
The project focuses on analyzing ride bookings, cancellations, revenue, customer behavior, ride distance, and driver/customer ratings to generate meaningful business insights.

The dashboard helps understand:

* Ride booking trends
* Revenue analysis
* Cancellation patterns
* Customer behavior
* Vehicle performance
* Operational efficiency

# 🧠 Business Insights

* Weekend bookings are significantly higher than weekdays.
* Prime SUV and Prime Sedan generate higher revenue.
* UPI is one of the most preferred payment methods.
* Driver cancellations are mainly due to personal and vehicle-related issues.
* Higher-rated drivers usually receive better customer ratings.
